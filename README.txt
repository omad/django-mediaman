========
Mediaman
========

Mediaman is a simple Django app to help manage images and media,
including tools for bulk uploading imag.es

It integrates with easy-thumbnails for generating thumbnails.

Detailed documentation is in the "docs" directory.

Quick start
-----------

1. Add "mediaman" to your INSTALLED_APPS setting like this::

      INSTALLED_APPS = (
          ...
          'mediaman',
      )

2. ...
